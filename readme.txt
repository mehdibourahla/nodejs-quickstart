PetApp is an application where people can find adorable pets like Cats dogs or Birds, and can send a request to the owner to adopt them.
So as User i can :
- Post an animal so other can see it and adopt it if they want.(POST)
- Scroll around all disponible animals.(LIST)
- Send a request to the owner.
- Chat with the owner.
- Post comments.(POST)
- Have more information about the owner/seeker.(SHOW)
- See animal details.(SHOW)
And ofcourse there must be :
- A register form.
- A login form.
- Profile Edit.
